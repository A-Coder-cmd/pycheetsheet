  2. BlackJack 2
    Question:- If the number is greater than 21 return 0 unless one of the numbers is 11. in such case the 11 should be converted to a 1 to prevent the sum from being exceeded. For example given a 11 and 13 as input, the 11 should be converted into a 1 so the total sum will be 14. 
    
# solution:-
# Taking two input a ,b form the user 
a = int(input("Enter a number:\n"))
b = int(input("Enter a another number:\n"))

def blackjack(a,b):
    """
    this function will take int a and b 
    if a or b is 11 than it will be conveted into 1 and then the number will be added
    """
    if a == 11:
        a = 1
        return a+b
    elif b == 11:
        b = 1
        return a+b
    else:
        return a+b

if __name__ == '__main__':
    run = blackjack(a,b)
    print(run)
  3. BlackJack Hand
    This is problem is scored like black-jack 2 ,but you need to deal with an array of numbers instead of one number: In general , return the sum of a given list of number if the sum is greter than 21, return 0 unless one or more of the numbers is 11 in such case as many 11's as necessary should be converted to 1's to prevent the sum from being exceeded 
    
# solution
BJ = [11,11,9,4,6]

def blackJackHand(numlist):
    SUM = 0
    for i in numlist:
        if i == 11:
            i = 1
            SUM = SUM+i
        else:
            SUM = SUM+i
    
    if SUM == 21:
        return 0
    else:
        return SUM

if __name__ == '__main__':
    run = blackJackHand(BJ)
    print(run)
